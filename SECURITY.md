# Security Policy

## Reporting a vulnerability

Use GitHub private vulnerability reporting: https://github.com/ebarahona/loopback-service-mongodb/security/advisories/new.

If GitHub reporting is unavailable, email ebarahona@gmail.com with the subject prefix `[SECURITY] @ebarahona/loopback-connector-mongodb`.

Do not file security reports as public issues.

## Supported versions

| Version range | Supported                                                                        |
| ------------- | -------------------------------------------------------------------------------- |
| Pre-1.0       | Latest minor only.                                                               |
| Post-1.0      | Latest major, plus the previous major for 6 months after the next major release. |

## Disclosure

- Reports are acknowledged within 7 days.
- Coordinated disclosure window is 90 days from acknowledgement, or the date of a coordinated public release, whichever is sooner.
- Researchers are credited by default in the published advisory. Credit is omitted on request.
