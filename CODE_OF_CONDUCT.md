# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Enforcement

Report violations to ebarahona@gmail.com.

## Reporting expectations

- Reports are handled confidentially.
- Reports are reviewed within 7 days of receipt.
- The reporter is informed of the outcome once the review is complete.
